# Qazpp
## Funkce
### Správa kvízů
- Vytváření nových
- Editace a mazání existujících
- Resetování
    - Smazání všech kvízů
    - Obnovení předpřipravených kvízů
### Vyhodnocování
- Splněný kvíz je na konci vyhodnocen
- Každá otázka = 1 správná odpověd
- Počet možných otázek a odpovědí na ně teoreticky nemomezený
